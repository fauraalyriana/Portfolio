ABC LEARNING CENTER

Project ini dikerjakan saat saya mengikuti bootcamp Lithan EduCLaas. Project ini bersifat individu, dimana kita diminta untuk merancang dan membuat website untuk ABC Learning Center dengan menggunakan HTML, CSS dan Javascript. ABC Learning Center adalah lembaga Pelatihan IT yang menyelenggarakan kursus seperti Pemrograman di Java, Pemrograman di .Net, Microsoft Office, HTML & dll. Website harus dapat digunakan untuk memberikan informasi tentang mereka kursus bagi siswa dan untuk menangkap prospek yang dihasilkan oleh pemasaran digital.

Dalam mengerjakan project ini, saya menggunakan HTML sebagai dasarnya. Kemudian dengan sentuhan CSS, saya mendekorasi situs untuk menciptakan pengalaman pengguna yang menyenangkan. Saya juga gunakan Javascript untuk menambahkan elemen interaktif ke situs.
