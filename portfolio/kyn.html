Know Your Neighborhood

Know your neighborhood adalah situs web yang memungkinkan pengguna untuk masuk menggunakan API yang ada dan sistem akan mengambil informasi dasar seperti nama email dari API.

alam membuat situs web "Know Your Neighborhood", saya memilih Visual Studio Code sebagai alat editor kode utama untuk mengembangkan seluruh proyek. Untuk mengembangkan backend, saya menggunakan Spring Tool Suite (STS) sebagai pilihan yang tepat. Kemudian, untuk mengakses API dan mengambil informasi dasar seperti nama dan email dari pengguna, saya menggunakan Postman. Dengan Postman, saya dapat membuat permintaan HTTP ke API yang ada dan melihat hasilnya, sehingga memastikan koneksi dan integrasi dengan API berfungsi dengan baik. Juga, dalam pengembangan front end, saya menggunakan React.js sebagai opsi untuk membuat antarmuka pengguna. Dengan menggunakan React.js, saya dapat membuat komponen UI yang dapat digunakan kembali, membuat pengembangan dan pemeliharaan frontend lebih mudah.

Proyek "Know Your Neighborhood" adalah kolaborasi yang menggabungkan teknologi canggih untuk menghasilkan situs web yang fungsional dan ramah pengguna. Visual Studio Code, Spring Tool Suite, dan Postman berperan penting dalam pengembangan backend yang solid dan akses ke API yang andal, sementara React.js digunakan untuk membuat antarmuka pengguna yang menarik dan interaktif.
