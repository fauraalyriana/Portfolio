WORD WIZARDRY

Word Wizardry adalah sebuah website yang didedikasikan untuk membantu orang-orang belajar dan meningkatkan kemampuan bahasa Inggris mereka. Project ini dibuat untuk memenuhi tugas akhir pada mata kuliah pemograman web saat berada di semester 5 universitas teknologi bandung. Project ini bersifat individu, dimana setiap mahasiswa bebas memilih tema website yang akan mereka kembangkan. Untuk bagian frontend saya menggunakan framework bootstrap, html, css serta javascript untuk membuat tampilan website. Lalu pada sisi backend saya menggunakan php.
