MangoCheck

mangocheck adalah website yang dapat digunakan untuk mengecek kematangan buah mangga. Website ini memungkinkan untuk pengguna mengunggah gambar buah mangga dan sistem akan menampilkan hasil apakah buah tersebut matang atau belum matang.

Projek ini dikerjakan secara kelompok saat mengerjakan Capstone Project di Bangkit Academy. Kontribusi saya pada projek ini yaitu pada penerapan Cloud Computing, Pembuatan Backend serta pembuatan antarmuka. Pada Backend saya menggunakan bahasa pemograman python agar memudahkan dalam konversi model Machine Learning yang telah dibuat oleh anggota lain. Pada frontend saya menggunakan framework react.js yang memungkinkan website menjadi responsif.

Projek ini juga saya lanjutkan sebagai tugas akhir kuliah saya yaitu skripsi, projek ini berhasil saya selesaikan saat skripsi dan mendapatkan nilai A.
